# blog

A blog system created with [Next.js](https://nextjs.org/), [Tailwind CSS](https://tailwindcss.com/), [MDX](https://github.com/mdx-js/mdx), etc.

## 使い方

```bash
npm install
npm run dev
```

`constants/data.ts`と`public`以下のファイルを適当にいじる

`pages/posts/[YYYYMMDD].mdx`に記事を書く(`pages/testPost.mdx`に全ての機能が書いてある)
